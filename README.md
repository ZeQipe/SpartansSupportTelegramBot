## Admin prompt update

Set two environment variables for prompt update via `/sys` command:

- `BOT_ADMIN_IDS` ‚Äì comma-separated Telegram user IDs allowed to update the prompt.
- `BOT_SYS_PASSWORD` ‚Äì password requested after `/sys`.

Sequence: `/sys` ‚Üí enter password ‚Üí send new system prompt. Old prompt is archived with timestamp, new one applied immediately without bot restart.
